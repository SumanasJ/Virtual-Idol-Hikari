# 📸 截图指南

## 如何为项目添加截图

### 推荐截图场景

#### 1. 对话界面 (主页面)
**展示内容**：
- 光的开场白或问候
- 多轮对话历史
- 贴纸消息展示
- 输入框和发送按钮

**建议尺寸**：800×500px
**截图时机**：打开应用后和光聊几轮，产生一些对话后

---

#### 2. 记忆系统 (光的小本本)
**展示内容**：
- 短期记忆列表（显示类型、重要性星级）
- 长期记忆列表
- 记忆编辑按钮
- "全部提升"按钮

**建议尺寸**：800×600px
**截图时机**：和光聊一段时间后，记忆列表有内容时

---

#### 3. 羁绊图谱
**展示内容**：
- 多个实体节点
- 关系连线
- 关系类型标签
- 羁绊卡片样式

**建议尺寸**：800×600px
**截图时机**：对话较多，羁绊关系丰富时

---

#### 4. 贴纸库
**展示内容**：
- 不同类型的贴纸（情感、美食、地标）
- 缓存统计信息
- 使用次数显示
- 预览效果

**建议尺寸**：800×600px
**截图时机**：生成了多个贴纸后

---

#### 5. 会话管理
**展示内容**：
- 多个会话卡片
- 当前会话标记
- 会话操作按钮（打开、删除）
- 会话信息（消息数、更新时间）

**建议尺寸**：800×600px
**截图时机**：创建多个会话后

---

### 截图工具推荐

#### Windows
- **Snipping Tool** (截图工具)
- **Win + Shift + S** (区域截图)
- **ShareX** (高级截图工具)

#### Mac
- **Cmd + Shift + 4** (区域截图)
- **CleanShot X** (推荐)

#### 浏览器扩展
- **Awesome Screenshot** (Chrome/Firefox)
- **Nimbus Screenshot** (支持标注)

---

### 图片优化建议

1. **尺寸**：宽度 800-1000px，高度自适应
2. **格式**：PNG (保持清晰度) 或 JPEG (文件更小)
3. **压缩**：使用 TinyPNG.com 优化
4. **命名**：
   - `screenshot-chat.png` - 对话界面
   - `screenshot-memory.png` - 记忆系统
   - `screenshot-graph.png` - 羁绊图谱
   - `screenshot-stickers.png` - 贴纸库

---

### 如何上传到 GitHub

#### 方法 1：直接上传到仓库

1. 将截图文件放到项目的 `screenshots/` 目录
2. 在 README.md 中引用：
   ```markdown
   ![对话界面](./screenshots/screenshot-chat.png)
   ```

#### 方法 2：使用 GitHub Issues

1. 在仓库创建新 Issue
2. 拖拽截图到评论框
3. 复制图片 URL
4. 在 README.md 中引用：
   ```markdown
   ![对话界面](https://user-images.githubusercontent.com/xxx/xxx/screenshot-chat.png)
   ```

#### 方法 3：使用图床

推荐图床服务：
- **Imgur** (免费)
- **Cloudinary** (开发者友好)
- **GitHub 自带的 Release 附件**

---

### 替换占位符图片

当前 README 使用的是占位符：
```markdown
![对话界面](https://via.placeholder.com/800x500/pink/white?text=聊天界面示例)
```

替换步骤：
1. 截取功能界面
2. 上传到图床或仓库
3. 替换 README 中的 URL
4. 提交更新

### 示例对话场景（用于截图）

#### 对话界面截图脚本
```
1. 打开应用
2. 光：呀吼~粉丝酱！最近怎么样？
3. 用户：我在学习你的项目代码
4. 光：哇！真的吗？好厉害！光都觉得有点不好意思呢~
5. 等待贴纸生成
6. 截图！
```

#### 记忆系统截图脚本
```
1. 和光聊10-15轮
2. 分享一些偏好（如：喜欢猫、爱喝咖啡）
3. 切换到"光的小本本"标签
4. 确保短期记忆和长期记忆都有内容
5. 截图！
```

#### 羁绊图谱截图脚本
```
1. 和光聊20轮以上
2. 聊不同话题（工作、爱好、食物）
3. 切换到"羁绊图"标签
4. 确保有多个节点和关系
5. 截图！
```

## 🎨 图片美化建议

### 添加标注
使用工具在截图上添加：
- 🔴 红圈标注重要功能
- 📝 文字说明
- ➡️ 箭头指示

### 边框装饰
```html
<div style="border: 3px solid #ec4899; border-radius: 12px; padding: 8px;">
  <img src="screenshot.png" alt="截图">
</div>
```

### 标注语言
建议添加双语说明：
- 🔥 **核心功能** / Core Feature
- 💾 **数据持久化** / Persistent Storage
- 🧠 **智能记忆** / Smart Memory

---

完成截图后，请：
1. 将图片放到 `screenshots/` 目录
2. 更新 README.md 中的图片链接
3. 提交 PR 或直接推送到 main 分支
